Updating AnalysisPlans.txt:

- In Excel, sort by analysisPlanSort, NAME1, NAME2, ..., NAME10
- If sort changed, update the RowId sequence column
- Delete all rows from the AnalysisPlan list on flow.labkey.org server (show all rows, select all on page, delete)
- Import entire list by uploading AnalysisPlans.txt file.
